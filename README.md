# ATM-fees
[![launch with diploi badge](https://diploi.com/launch.svg)](https://diploi.com)

Contains the original spreadsheet that the Revolut's community created back when Revolut had a public forum.

I created a tool using the original data from the spreadsheet, where you can check each country's ATM fees when you want to withdraw cash.

https://atmfees.dev

You can suggest changes or add new ATM fees by creating an issue or a PR.
